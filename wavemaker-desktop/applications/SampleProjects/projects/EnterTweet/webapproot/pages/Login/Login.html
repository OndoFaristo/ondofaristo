EnterTweet Application

This application demonstrates WaveMaker's ability to build a SaaS-enabled, multi-tenant application in under 10 minutes.

It integrates the Twitter web services API with a contact database.

To login, try the following username/password combinations:
  • Username: user, password: user [tenant 1 user]
  • Username: admin, password: admin [tenant 1 admin]
  • Username: user2, password: user2 [tenant 2 user]
  • Username: admin2, password: admin2 [tenant 2 admin]

This application was built using the enterprise version of WaveMaker that includes role-based access controls and multi-tenant security. For more information, go to www.wavemaker.com.

Note: this is a sample application and the database is refreshed regularly.
