Welcome to IssueCloud

This application demonstrates how to build multi-tenant cloud applications using WaveMaker.

This application is hosted on Amazon EC2 and uses WaveMaker's multi-tenant security to automatically enforce data isolation between companies or tenants. Issuecloud demonstrates email workflow, enabling the browser back button and other advanced WaveMaker techniques.
